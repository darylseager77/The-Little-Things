## ðŸ› Troubleshooting

**Problem: Netlify deploy failed**
- Solution: Make sure you only uploaded the 3 files (html, css, js)
- Try zipping them first, then upload the zip

**Problem: Site looks broken**
- Solution: Clear your browser cache (Cmd+Shift+R or Ctrl+Shift+R)
- Check that all 3 files uploaded

**Problem: Buttons don't work**
- Solution: Check browser console for errors (press F12)
- Make sure script.js uploaded correctly

**Problem: Want to update the prototype**
- Solution: Make changes to the files locally
- Zip them again
- Drag to Netlify (it will update automatically)
